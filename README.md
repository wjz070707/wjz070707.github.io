# Jiaze Wang Personal Website

This repository hosts the source code for Jiaze Wang's personal academic website:

https://wjz070707.github.io

The site is built with Jekyll and published through GitHub Pages.

## Local Preview

Install the Ruby dependencies, then run:

```bash
bundle exec jekyll serve -l -H localhost
```

The site will be available at:

```text
http://localhost:4000
```
